
## Features Demonstrated

### ✅ Enhanced GitHub Theme Support

- **Light Mode**: Uses `github-light` theme with clean, readable syntax highlighting
- **Dark Mode**: Uses `github-dark-dimmed` theme for reduced eye strain
- **Automatic Switching**: Themes change automatically when you toggle the site theme
- **GitHub-Inspired Colors**: Line and word highlighting match GitHub's color scheme
- **Accessibility**: High contrast and reduced motion support included

### ✅ Line Numbers

- Add `showLineNumbers` to any code block meta string
- CSS counters provide consistent numbering
- Properly styled for both light and dark themes

### ✅ Line Highlighting

- Use `{1,3-4}` syntax to highlight specific lines
- Highlighted lines have subtle background and left border
- Works with line numbers

### ✅ Word Highlighting

- Use `/word/` syntax to highlight specific words or patterns
- Highlighted words have background color and ring
- Can combine multiple patterns: `/useState/ /count/`

### ✅ Code Titles

- Add `title="filename.tsx"` to show file names
- Titles are styled consistently with the code block
- Rounded corners connect title to code block

### ✅ Language Support

- TypeScript/TSX, JavaScript/JSX
- Bash/Shell scripts
- CSS, JSON, and more
- Proper syntax highlighting for each language

## GitHub Theme Integration

The code highlighting leverages GitHub's official themes for authentic syntax highlighting:

### Available GitHub Themes

- **github-light**: Standard GitHub light theme
- **github-light-default**: GitHub's default light theme
- **github-light-high-contrast**: High contrast light theme for accessibility
- **github-dark**: Standard GitHub dark theme
- **github-dark-default**: GitHub's default dark theme
- **github-dark-dimmed**: Softer dark theme (currently used) for reduced eye strain
- **github-dark-high-contrast**: High contrast dark theme for accessibility

### Integration Features

- **Seamless Theme Switching**: Automatically adapts to site theme changes
- **Enhanced Typography**: Supports font ligatures and advanced font features
- **GitHub-Inspired Highlighting**: Line and word highlights match GitHub's color scheme
- **Accessibility Support**: High contrast and reduced motion preferences respected
- **Backdrop Effects**: Subtle blur and transparency for modern appearance
- **Smooth Transitions**: Animated theme changes for better user experience

### Customization

The theme configuration can be easily customized in `src/lib/shiki.ts`:

```tsx showLineNumbers title="src/lib/shiki.ts"
export const REHYPE_PRETTY_CONFIG = {
  theme: {
    light: "github-light",
    dark: "github-dark-dimmed", // Easy to change
  },
  // ... other options
};
```
